# 欲求ふりかえりアプリ

このアプリは、人間の様々な欲求をチェックし、自分の状況を振り返るためのツールです。「人間の欲求リストCSV」を元に作成されており、簡単な入力フォームからデータの可視化まで行えます。

## デモ
[欲求ふりかえりアプリ](https://code4fukui.github.io/human-desires/)

## 機能
- 人間の80種類以上の欲求をチェックできる
- チェック結果をグラフで可視化できる
- チェック状況をURLにハッシュ化して共有できる
- 未回答の設問にジャンプできる
- 回答をリセットできる

## 必要環境
特にありません。ウェブブラウザさえあれば利用できます。

## 使い方
1. ウェブサイトにアクセスする
2. 各設問に回答する
3. 回答が完了するとQRコードが表示される
4. QRコードをスキャンするか、URLをコピーして共有する
